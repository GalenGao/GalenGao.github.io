# Installing Nmon  
For `x = 234`, the output should be `ReverseBit(x) = 87`.
`23410 = 111010102 => 010101112 = 8710`.  
**Example**  
* **[input] integer x**  
* **[input] integer x**  

```python
def ReverseBit(x):
  x = bin(x).replace('0b', '')
  reverse_text = ''
  for l in range(len(x)-1, -1, -1):
      reverse_text = reverse_text + x[l]
  return int(reverse_text, 2)
```
